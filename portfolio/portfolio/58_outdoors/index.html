Clare's GIS Portfolio

Outdoors

Hills around Loch Torridon, Highlands of Scotland

Data

  • Background mapping - places, water, roads etc, from OpenStreetMap.
  • Elevation is SRTM data from NASA downloaded via SRTM downloader plugin in QGIS.

Tools

  • All done in QGIS.

What would I like to change?

Originally intended to do a map fo the coire of Beinn Alligin with photos and highlights from our explorations. Was taking far too long to set it up, so I used the background data and elevation to produce a quick map of the hills.

So, I’d like to develop the map of the coire of Beinn Alligin.

Process

  • Elevation data downloaded via plugin. Merged and reprojected then coloured with an elevation colour ramp.
  • Elevation layer copied and styled as hillshade - multiply blending mode and transparency so it isn’t overwhelming.
  • Data downloaded from OSM via the QuickOSM plugin and styled with my usual stylesheets.
  • Created a layout, added title, text, scalebar.
  • Exported as map.
