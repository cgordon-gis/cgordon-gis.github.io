Torridon Land Cover

Data

  • UKCEH Land Cover 2019 WMS
  • COP30 DSM - European Space Agency (2024). Copernicus Global Digital Elevation Model. Distributed by OpenTopography. https://doi.org/10.5069/G9028PQB. Accessed 2025-11-01
  • Hill names from OpenStreetMap via QuickOSM plugin.

Tools used

  • QGIS.
  • Inkscape.

What did I learn?

  • Practice at altering labels with the advanced labelling toolbar.

What would I do differently or want to find out?

  • Make the colours/key more accessible - label land cover areas and add labels to key.
  • Layout too crowded? Think about it further.
  • Try out functions on annotation toolbar in QGIS to change how labels work. e.g. Would be good to have main mountain names following the ridges.

Process (more detail about how I made the map)

  • Add WMS layer to QGIS and add to map.
  • Add COP30 DSM and style as hillshade. Use Multiply blend mode.
  • Use QGIS plugin QuickOSM to add features where natural = peak.
  • Label peaks with name. Then used Label toolbar to move and restyle individual labels. Also added new features for overall hill names, e.g. Liathach, and styled those as semi-bold.
  • Used template to set up layout then added text and image.
  • Exported as image (600 dpi) and imported into Inkscape to add key to landcover as WMS landcover key doesn’t output from QGIS.
  • Export from Inkscape as image.
