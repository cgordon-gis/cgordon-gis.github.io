Air

North Sea windspeed and direction animation

Inspiration from Visualising Wind from a GRIB Mesh Layer in QGIS - YouTube video from Hans van der Kwast.

Data

  • Downloaded data for North Sea from: https://www.euroszeilen.utwente.nl/weer/grib/ (Creative commons licence.)
  • Natural Earth Data country outlines from: https://www.naturalearthdata.com/downloads/10m-cultural-vectors/ - (Acknowledgement: Made with Natural Earth. Free vector and raster map data @ https://naturalearthdata.com.)

Tools used

  • QGIS to create the map and export the animation frames.
  • GIMP to combine the frames to create the animated gif.

What did I learn?

  • How to style a mesh file with arrows and make the arrows spaced out.
  • How easy it is to animate this data!

I’ve created animated gifs in GIMP before, so that wasn’t new for me.

What would I do differently or want to find out?

  • Try and make the output smaller!
  • Can I find open wind data for the UK? Investigate Copernicus ECMWF.
  • I’d also like to use historic data, e.g. from a storm rather than just a quiet day!
  • Try out Traces in QGIS too - I didn’t have time this time.

Process (more detail about how I made the map)

  • Added mesh layer from .grb file.
  • Added country outlines from Natural Earth.
  • Using mesh styling - Surface wind gusts styled as Contours and wind [m/s] as vector arrows.
  • Set up animation as usual.
  • Exported images.
  • Created animation in GIMP.
