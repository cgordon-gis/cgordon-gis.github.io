Points 3

Public bookcases (little libraries) in West Yorkshire

1st November 2025

Data

  • OSM data for West Yorkshire downloaded from Geofabrik as .osm.pbf
  • Other OSM data downloaded via the OSMInfo and QuickOSM plugins in QGIS.

Tools used

  • GDAL
  • QGIS

What did I learn?

  • First go with new commands in GDAL application at the command line. Will need to get used to them, but seem so much more consistent than “traditional” commands.
  • Trying out clustered points in QGIS. Probably work best with larger numbers?

What could I do differently or want to find out?

  • It would be interesting to know how the locations of little libraries matches demographics of the areas where they do or don’t occur.
  • I’ve done a rather similar style to my first map in 2023. Could I come up with something more imaginative?

Process

Preparing data

  • Trying out new commands for GDAL in version 3.11 - using via command line.

To find out layers in osm.pbf file:

>>> gdal vector info -i west-yorkshire-251031.osm.pbf -f text`

Extract little libraries using sql. OSM tag is public_bookcase.

>>> gdal vector sql rawdata\west-yorkshire-251031.osm.pbf proc\WestYorkshire.gpkg --output-layer public_bookcase --sql="SELECT * FROM points WHERE other_tags LIKE '%public_bookcase%'"

Then reproject to British National Grid.

>>> gdal vector reproject --src-crs=EPSG:4326 --dst-crs=EPSG:27700 proc\WestYorkshire.gpkg --layer=public_bookcase --append proc\WestYorkshire.gpkg --output-layer=public_bookcase_27700

Styling data in QGIS and preparing a layout

  • Add public bookcase layer and style with SVG book symbol. Then change style to a point cluster.
  • Use OSMInfo plugin to download polygon for West Yorkshire. Styled polygon as dark grey.
  • Set map properties background to black.
  • Used QuickOSM plugin to download Cities and Towns. Styled as labels with no point. Used Move a label tool to move some labels away from book symbols, then deleted some towns as too many on map.
  • Set up a layout and exported as image.
