Retro

Bishop’s Castle Pubs

Data

  • OpenStreetMap via QuickOSM plugin - buildings, historic and highways
  • Font: BellTopo Sans from Sarah Bell (add link)

Tools

  • QGIS

Process

  • Download OSM layers.
  • Extract current pubs from buildings label as a new layer.
  • Labelled current and historic pubs.
  • Fiddled around endlessly with styles in an attempt to get the map looking retro!
  • Added a background via Project > Properties and chose turquoise/blue.

What would I change or would like to improve?

  • Not happy with this, but not sure how I could improve it!
  • Haven’t really used any new to me techniques, except maybe using callouts for the labels, which I haven’t often done. But doing the styling has stretched me!
