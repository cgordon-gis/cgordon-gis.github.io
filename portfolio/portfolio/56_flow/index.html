Flow

Flow of Northern Gannets foraging from Alderney

Creating an animated gif from multiple frames.

Data

  • Alderney Gannets tracking data from Movebank.

Tools

  • Generated maps of route with Time Manager in QGIS. QGIS tutorials - Animating Time Series Data
  • Used GIMP to create an animated GIF.

What did I learn?

  • Setting up the Temporal Controller in QGIS.
  • Exporting frames with the Temporal Controller.
  • More practice on how to create an animated gif in GIMP!
  • Keeping file sizes right down with page size and resolution.

Process

  • Add point data to QGIS. Check for time/date column. In this case it is timestamp.
  • Open layer properties and go to Temporal tab on left.
    • Select Configuration as Single Field with Date/Time.
    • Limits = Include start, Exclude End (default)
    • Set the Field correctly - to timestamp for this data.
    • Set Event duration as required. You can always change this later!
    • Choose whether to Accumulate features over time - can get very messy if you have a lot of points!
    • Click Apply and OK.
  • Go to View > Panels > Temporal Controller or click the button on the toolbar.
  • On the panel, click on the button with the green arrow and a tooltip of Animated temporal animation.

This should give you some more controls. The Animation range should be set by default. Try clicking on the Play button and see what happens.

  • Change the Step setting, e.g. to 1 hour or 1 day, depending on the time period of your data, or how quickly it changes. For the Alderney Gannets, 30 minutes worked best.
  • To save images to be built into an animation, once you’re happy with the animation click on the save button next to Step.
    • Fill in the form as requried, particularly with Extent - don’t make it too large!
    • You can also control the temporal settings from this dialogue.
    • Decide where you want to save the multiple generated images (one per movement frame).
    • When happy, click on Save and wait for the images to be generated.

You should end up with multiple images in a single folder. You now need to use GIMP for the next stages.

Creating animation

Used GIMP to create an animated GIF.

  • Create a new canvas - File > New then enter correct size (refer to pixel size of images) and orientation. Set resolution under Advanced - don’t make it too high! 95 dpi should be fine.
  • Add images as “frames” starting with the first image at the “bottom” of the layers.
    • Can just drag and drop images one at a time from the bottom onto the canvas (not the layer panel!) and each one is added as the next layer up.
    • Delete background layer.
  • Save!
  • To keep size down you can use an indexed colour mode instead of RGB. Image > Mode > Indexed. Stick to defaults for now!
  • File > Export As - give file name .gif extension then click Export.
  • In dialog that opens, select As animation and choose options.
    • Untick GIF comment:, unless you want to add a comment of your own…
    • Frame disposal should be set to One frame per layer (replace)
    • Tick both Use delay entered above for all frames and Use disposal entered above for all frames.
    • Change Delay as required to change speed of animation.
  • Click Export to export the animated gif.
  • Open in a graphics program to check and if necessary, Export from GIMP again, changing Delay as required to speed up or slow down your animation.
