10 minute map

UK Earthquakes over magnitude 3 since 1970

Data

  • ESRI Living Atlas - BGS UK Earthquakes
    • Data acknowledgement: Contains British Geological Survey materials Â©UKRI 2025.
    • From information on Living Atlas page: “ML: Richter local magnitude of the event (size of the earthquake)”.

Tools used

  • QGIS.

What did I learn?

It’s really difficult to make even a relatively simple map in 10 minutes.

What would I do differently or want to find out?

  • Take more time to have a look at the data.
  • Style as graduated size and colour? Or some way of showing time element?

Process (more detail about how I made the map)

  • Used URL from Living Atlas page.
  • Set up connection as ArcGIS REST Server and added layer to QGIS.
  • Used Select by expression to select records with ML >= 3.
  • Style as graduated size - Pretty breaks.
  • Create a layout. Add key, title and other text and use dropper to set all text to similar blue to data.
  • Export as image.
