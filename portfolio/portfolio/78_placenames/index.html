Places and their names

Gaelic colour place names around Loch Torridon, N.W. Scotland

Data

  • Useful list of colours at https://learngaelic.scot/fichead-facal/ff-colours.jsp
  • Data from Torridon place names

Tools used

  • QGIS.

What did I learn?

  • A reminder of some Scottish Gaelic colour names.
  • Interested to see only two placenames near Torridon include “uaine”.

What would I do differently or want to find out?

  • Spend more time placing the labels.
  • A bit more work on the exact colours to make them more visible or change the background.

Process (more detail about how I made the map)

  • Downloaded as kml file.
  • Add points layer to QGIS. Export as GeoPackage with crs = EPSG:27700.
  • Rule-based styles - style for each of Gaelic colours used. Use ILIKE in filter as case insensitive. e.g.
"Name" ILIKE '%dubh%'
  • Set each point symbol to a different shape - will match in key and help with colour identification.
  • Rule-based labelling. Set up a rule for each colour. Set each label colour to match colour of point using dropper tool.
  • Background map: World Ocean Base from ESRI Living Atlas.

Colours used:

  • dubh = black
  • glas = grey
  • dearg = red
  • ruadh = red / brown
  • liath = light blue / grey
  • geal = white
  • buidhe = yellow
  • gorm = dark blue
  • uaine = green
