GPS tracks - exploring Kinlochewe

Walks we did around Kinlochewe on holidays from 2023 to 2025.

Data

  • GPX files from tracks recorded myself
  • Topo background map from MapTiler Plugin in QGIS.

Tools used

  • Locus Map for recording gps tracks
  • GDAL to process folder of gpx files via a python script.

What did I learn?

  • A quick script to batch convert a folder of files with python and GDAL.
  • Where to change feature rendering order for a layer in QGIS.

What could I do differently or want to find out?

  • Merge all tracks into a single GeoPackage as I convert them.
  • A clearer way of showing where tracks overlap.
  • Add a scale bar!
  • Add a graph showing lengths of walks.

Process

  • Download gpx files via Locus mobile app

Batch converting folder of gpx files to GeoPackage. Not time to work out how to add all to same GeoPackage, so used this as python script - based on SpatialThoughts Mastering GDAL Tools - Running commands in batch.

import os

input_dir = 'data'
output_dir = 'proc'

command_string = 'gdal vector convert --input={input} --layer=tracks --output-format=GPKG --output={output}'
for file in os.listdir(input_dir):
  if file.endswith('.gpx'):
    input = os.path.join(input_dir, file)
    filename = os.path.splitext(os.path.basename(file))[0]
    output =  os.path.join(output_dir, filename + '.gpkg')
    command = command_string.format(input=input, output=output)
    print('Running ->', command)
    os.system(command)
  • Added all resulting gpkgs to QGIS, then merged vector layers.
  • Added length in km to each track using Field calculator - round($length/1000, 2).
  • Styled as graduated style, pretty breaks.
  • Added Draw effect drop shadow.
  • Changed feature rendering order so shorter tracks display on top of longer tracks.

Creating a background
