Raster

Bathymetry of the Sound of Raasay and the Inner Sound

Playing with height data again. Always fun and interesting! This time combining elevation and bathymetry.

Data

  • Bathymetry data downloaded from: https://www.emodnet-bathymetry.eu/
  • SRTM data downloaded via SRTM Download plugin in QGIS

Tools

  • QGIS for data handling and setting up the layout.
  • Inkscape for labelling.

What would I like to change?

  • Would like to try doing this as a plot using R. Thought about it this time, but it would have taken too long.
  • Would this benefit from a key? As it is, it is decorative, but a key would make it a bit more informative…

Process

  • Downloaded bathymetry and elevation data. Both needed to be merged then reprojected to British National Grid.
  • Styled each individually as singleband pseudocolour with appropriate colour ramps.
  • Duplicated both layers and styled the copies as hillshade with Multiply blending mode, and quite a bit of transparency so they’re not too obvious!
  • Set up a layout for the area I wanted, added title etc, scalebar and extent indicator.
  • Exported layout as image.
  • Imported into Inkscape and added labels using spacing between letters, and text on path for the Sound of Raasay label.
  • Finally, exported as a png image.
