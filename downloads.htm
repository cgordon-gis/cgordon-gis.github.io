GIS downloads

Contents

Please note that these files are provided 'as is'. My priority is to the students and staff of the School of Earth and Environment in Leeds and I am unable to spend much time in supporting external users.

If you are downloading .zip or .7z files, right-click and save to your local drive (preferably your M: drive on computers in University of Leeds clusters), then unzip by right-clicking and extract (preferably with 7-zip) before you use the contents in GIS. Note that 7-zip has proved much more reliable for unzipping files in the past than Extract All.

Workbooks

The date of latest update is given on the front cover of each individual workbook. Some workbooks include a list of the latest updates immediately after the contents list.

If you are a member of the University of Leeds and wish to have a copy of the data that I provide for the students on my courses to go with the workbook, please email me at c.e.gordon@leeds.ac.uk and let me know which workbook you are interested in.

QGIS

Since 2020 I have been teaching GIS with QGIS. These workbooks are the ones I am using for my current courses.

  • Environmental Science workbook (Word docx format).

ArcGIS

  • Environmental Science workbook (ArcGIS Pro) - ArcGIS Pro version for 2019. A general introduction to GIS using Environment Science examples. No longer being updated.
  • Environmental Science workbook (ArcMap) - ArcMap version for prior to 2019. A general introduction to GIS using Environment Science examples. No longer being updated.
  • Geological Sciences workbook for ArcGIS Pro - used for 2019 and 2020. A basic workbook showing how to draw a geological map in GIS.
  • Geological Sciences workbook for ArcMap - In use prior to 2019. This is no longer being updated. A basic workbook showing how to draw a geological map in GIS.
  • Geophysics / Meteorology workbook. A short introduction to basic GIS.
  • Supplementary exercises for geological science students for ArcGIS Pro - use for 2020. A brief introduction to some further techniques with geological examples.
  • Digimap workbook

Inkscape

  • Vector graphics (Inkscape) interactive worksheets for geological sciences - these open in Inkscape and the exercises are carried out within the worksheet. They cover the main techniques for producing scientific diagrams and map/poster layouts in Inkscape. (Zip file)
  • Vector graphics (Inkscape) quick reference guide - to support the interactive exercises above. (Also included in the Interactive Inkscape zip file.)

Extra information sheets

  • Using Open Source GIS: Freely available data and software. These are fairly rough notes rather than a complete workbook.

Miscellaneous

  • Geological symbols in png and svg formats - A small selection. Should work with both ArcGIS and QGIS. (Zip file - unzip before using.)
  • QGIs styles for OpenStreetMap - work with data converted for QGIS in way shown in Open Source GIS instructions above. (Zip file)
  • Right hand rule example Excel spreadsheet - linked from Geological Sciences workbook.
  • Example gpx file for Beckett Park - sample data for use with geological mapping workbook. (Right-click to download.)
